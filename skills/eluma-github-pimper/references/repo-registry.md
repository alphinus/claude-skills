## Footer Template

Use this in every README footer:

```markdown
<div align="center">
<sub>

**More from [alphinus](https://github.com/alphinus):**
[claude-skills](https://github.com/alphinus/claude-skills)

</sub>
</div>
```

## Adding a New Repo

1. Add entry to this registry
2. Update footer template with new repo link
3. Update ALL existing repo READMEs with new footer
4. Generate social preview for new repo
