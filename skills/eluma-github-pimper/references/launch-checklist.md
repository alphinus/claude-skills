# Repo Launch Checklist

## Pre-Launch (T-7 days)

### Repo Polish
- [ ] README is complete with banner, badges, screenshots
- [ ] Social preview image uploaded (Settings → Social preview)
- [ ] CONTRIBUTING.md is in place
- [ ] Issue templates are set up
- [ ] License file exists
- [ ] .gitignore is clean (no .DS_Store, node_modules, .env)
- [ ] All links in README work (check with `markdown-link-check`)
- [ ] Screenshots are current and high-quality
- [ ] Quick Start actually works (test on fresh machine)

### Content Prep
- [ ] Write HackerNews post title (technical, no marketing speak)
- [ ] Write ProductHunt tagline (benefit-oriented)
- [ ] Prepare 3 YouTube Shorts scripts
- [ ] Record screen recordings for Shorts
- [ ] Draft dev.to / hashnode blog post
- [ ] Prepare Twitter/X thread (5-7 tweets)

### Community
- [ ] Identify 3-5 relevant subreddits
- [ ] Find relevant Discord servers
- [ ] Prepare answers for FAQ

## Launch Day (T-0)

### Morning (6-8am US time)
- [ ] Post on HackerNews (technical title, no exclamation marks)
  - Title formula: "Show HN: [Name] — [what it does]"
  - Average impact: 121 stars in 24h, 289 in one week
- [ ] Post on relevant subreddits
- [ ] Share on Twitter/X with GIF/video

### Midday
- [ ] Launch on ProductHunt (if applicable)
- [ ] Post on dev.to / hashnode
- [ ] Share in relevant Discord servers
- [ ] Engage with ALL HN comments (fast response = more upvotes)

### Evening
- [ ] Post first YouTube Short
- [ ] Reply to all GitHub issues/stars
- [ ] Cross-post to LinkedIn

## Post-Launch (T+1 to T+7)

- [ ] Post remaining YouTube Shorts (1 every 2 days)
- [ ] Write follow-up blog post with learnings
- [ ] Thank top contributors
- [ ] Create "v1.1" milestone with community-requested features
- [ ] Update star-history badge if crossed milestone
- [ ] Post "Week 1 results" thread on Twitter/X

## Metrics to Track
| Metric | Tool |
|--------|------|
| Stars over time | star-history.com |
| Traffic sources | GitHub Insights → Traffic |
| Clone count | GitHub Insights → Traffic |
| Unique visitors | GitHub Insights → Traffic |
| HN rank | news.ycombinator.com |
| Reddit upvotes | old.reddit.com |
| YouTube Short views | YouTube Studio |

## Cross-Repo Promotion
After launch, update ALL other repo READMEs to include this new repo in the footer cross-links section.
