# Brand Machine — Prompt Templates

Curated prompts for generating brand assets via AI image models (FLUX, Gemini, etc.).

## Variables
Replace these placeholders with actual brand values:
- `{BRAND_NAME}` — Brand display name
- `{PRIMARY}` — Primary hex color
- `{MOTIF}` — Design motif (geometric, hexagonal, organic, minimal)

## Tips for Best Results

1. **Be specific about composition** — "centered", "left-aligned", "with whitespace"
2. **Include color context** — Mention both primary color and background
3. **Specify style** — "vector", "minimalist", "professional", "tech"
4. **Add quality markers** — "high quality", "crisp edges", "clean lines"
5. **Set aspect ratio** when needed for specific platforms
